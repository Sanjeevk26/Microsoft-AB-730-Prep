# Creating Effective Prompts in Microsoft 365 Copilot

## 1. Overview

Microsoft 365 Copilot allows users to create prompts across multiple interfaces, including:

- Copilot Chat (via office.com → m365.cloud.microsoft)
- Microsoft 365 applications (Word, Excel, PowerPoint)
- Agents (specialized tools)
- Creation tools (images, documents, presentations, etc.)

Users can enhance prompts by attaching:
- Work content (SharePoint, OneDrive)
- Files and images
- Cloud documents
- Voice input (microphone)

---

## 3. Basic Prompt Example

A simple prompt:

"Explain how to calculate Gross Domestic Product"

This will generate a response, but it may not be focused or tailored.

---

## 4. Structured Prompting Approach

An effective prompt should include the following components:

### 4.1 Goal (What)
Defines the task clearly

Example:
"Identify key factors contributing to GDP"

---

### 4.2 Context (Why)
Explains the purpose or audience

Example:
"For a board-level presentation"

---

### 4.3 Expectations (How)
Defines format, tone, and structure

Example:
- Use a professional tone
- Provide step-by-step explanation
- Keep it concise

---

### 4.4 Source (Using What)
Specifies data sources

Example:
"Use BBC News as the primary source"

---

## 5. Prompt Structuring Template

A well-structured prompt typically follows:

- Goal (What to do)
- Context (Why it is needed)
- Expectations (How it should be delivered)
- Source (What data to use)

---

## 6. Example Prompts

### 6.1 Example 1: Summary Task

Summarize the main points from the Q4 sales report in SharePoint.  
Include key figures and trends.  
Present the summary as bullet points and keep it under 200 words for a senior management presentation.

---

### 6.2 Example 2: Data Analysis

Analyze the Q4 sales data in Excel and provide a summary of the top three performing regions.  
This is for a quarterly performance report.  
Include total revenue and percentage growth compared to Q3.  
Present the results in a table format.

---

### 6.3 Example 3: Email Draft

Draft a professional email to the finance team requesting approval for the updated budget proposal.  
Keep the tone formal and include a summary of changes.  
Attach the proposal document.

---

### 6.4 Example 4: Meeting Agenda

Create a meeting agenda for the upcoming project kickoff session.  
Include objectives, key discussion points, and time allocation for each topic.  
Present it in a clear and structured format.

---

### 6.5 Example 5: Executive Summary

Generate an executive summary of the annual performance report.  
Highlight revenue growth, major achievements, and key challenges.  
Keep it under 300 words and use a professional tone for leadership review.

---

## 7. Prompt Formatting Tips

- Use bullet points for multiple instructions
- Use hyphens (-) to list sources or inputs
- Use Shift + Enter to create new lines without submitting the prompt
- Avoid vague instructions
- Use clear and positive instructions (e.g., "Write this" instead of "Do not write this")
