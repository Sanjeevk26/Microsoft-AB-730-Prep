# Creating Custom Agents in Microsoft 365 Copilot

## 1. Overview

In addition to using pre-built agents, Microsoft 365 Copilot allows users to create **custom agents** when existing agents do not meet specific requirements.

Custom agents enable:
- Integration with external systems
- Use of internal and external knowledge sources
- Custom instructions and behavior
- Task-specific automation

---

## 2. When to Create Custom Agents

You should create a custom agent when:

- Built-in agents do not provide required functionality
- You need domain-specific intelligence
- You want to integrate external data sources
- You require customized behavior or tone

---

### 3.2 Surgical Planning (Healthcare)

---

### 3.3 Employee Onboarding Assistant

- Answers FAQs
- Guides users through processes

---

## 4. Copilot Studio Lite vs Full

### Copilot Studio Lite

- Simplified agent creation
- Supports:
  - Instructions
  - Knowledge sources
  - Capabilities

---

### Limitations

- No workflows
- No structured dialogues
- No advanced automation

---

### Copilot Studio Full

- Supports:
  - Topics (conversation flows)
  - Conditions and variables
  - Workflow automation (e.g., Power Automate)
  - Integration with external services

---

## 5. Creating an Agent Using Templates

### Steps

1. Click **Create Agent**
2. Choose between:
   - Describe tab
   - Configure tab
3. Select a template
4. Customize the agent
5. Test the agent

---

## 7. Customizing an Agent

After selecting a template:

- Modify instructions
- Add tone and style preferences
- Define output expectations

---

### Example Customization

- Use a professional tone
- Address the user in a specific way
- Adjust response format

---

## 8. Testing the Agent

After configuration:

- Use suggested prompts
- Provide context when required

---

### Example Prompt

"Provide technical interview questions for a Microsoft Copilot AI Business Professional role"

---

### Follow-Up Inputs

- Job description
- Number of questions
- Preferred format

---

### Output

- Structured interview questions
- Multiple formats
- Relevant to the specified role

---

## 9. Key Capabilities of Custom Agents

- Use internal and external knowledge sources
- Provide structured responses
- Adapt to specific use cases
- Enable reusable workflows (limited in Lite)

---

## 10. Best Practices

- Start with templates for faster setup
- Customize instructions clearly
- Validate outputs before use
- Use appropriate data sources
- Upgrade to Copilot Studio Full for complex scenarios

---

## 11. Key Takeaways

- Custom agents provide tailored AI experiences
- Templates simplify agent creation
- Copilot Studio Lite supports basic customization
- Advanced workflows require Copilot Studio Full
- Agents can integrate with enterprise and external data sources
