# Microsoft 365 Copilot Chat and Copilot Experience

## 1. Signing into Microsoft 365 Copilot Chat

To access Microsoft 365 Copilot Chat:

1. Go to: https://www.office.com  
2. Sign in using your Microsoft account  
3. Complete multi-factor authentication if prompted (e.g., Microsoft Authenticator)  
4. Navigate to: https://m365.cloud.microsoft  

This will open the Microsoft 365 Copilot Chat interface.

---

## 2. Overview of Microsoft 365 Copilot Chat Interface

Once logged in, the Copilot Chat interface provides the following components:

### 2.1 Navigation and Access

- Home (Copilot Chat landing page)
- Search functionality for quick access
- Chat history (previous conversations)
- Agents (specialized tools with defined instructions)
- Pages, images, and notebooks library

---

### 2.2 Creation Capabilities

Users can create:
- Images
- Videos
- Documents and structured outputs

---

### 2.3 App Access

The interface provides direct access to web versions of:
- Microsoft Word
- Microsoft Excel
- Microsoft PowerPoint

There is also access to:
- Microsoft 365 Admin Center

---

## 3. Work vs Web Mode

At the top of the interface, users may see:

- Work tab
- Web tab

### 3.1 Work Mode
- Available only if the user has Microsoft 365 Copilot (Business or Enterprise)
- Responses are grounded in:
  - Organizational data (SharePoint, OneDrive, Teams, Outlook)
  - Web data

### 3.2 Web Mode
- Available to all users
- Responses are grounded only in:
  - Web data
  - Uploaded files

### Key Observation

If a user does not have a Copilot license:
- Work tab is not visible
- Only Web mode is available
- A prompt to upgrade Copilot may appear

## 4. Copilot Chat Behavior

- "What is the weather tomorrow in London?"
- "Summarize this document"

### Response Sources

| Mode        | Data Source |
|-------------|------------|
| Web         | Internet + uploaded files |
| Work        | Internet + uploaded files + enterprise data |

## 10. Microsoft 365 Copilot Inside Applications

Copilot is also embedded directly into Microsoft 365 apps.

### Example: Microsoft Word

When opening a document:
- A Copilot prompt appears at the top
- A Copilot button is available in the Home tab
- Opens a Copilot side pane

### Capabilities:
- Generate content
- Summarize documents
- Rewrite or enhance text

---

## 11. Switching Licenses in Microsoft 365 Apps

To change the active license:

1. Go to File  
2. Select Account  
3. Click Switch License  

This determines whether:
- Copilot features are enabled
- Full enterprise capabilities are available

---

## 12. Key Differences: Copilot vs Copilot Chat

| Feature                | Copilot (Apps)            | Copilot Chat (Web)        |
|------------------------|---------------------------|----------------------------|
| Location               | Inside Office apps        | Browser-based interface    |
| Current file context   | Yes                       | No                         |
| Work data grounding    | Yes (licensed)            | Yes (licensed only)        |
| Use case               | Productivity workflows    | General queries and tasks  |

---

## 13. Best Practice Guidance

- Use Web mode for general queries unrelated to work
- Use Work mode for enterprise-specific queries
- Use Copilot in apps for document-centric tasks

---

## 14. Learning Recommendation

Consistent learning improves retention and outcomes.

- Schedule regular learning sessions
- Build hands-on familiarity with Copilot features
- Practice switching between Work and Web modes
- Explore Copilot in different applications

---

## 15. Summary

- Microsoft 365 Copilot Chat is a web-based AI interface
- Work mode enables access to enterprise data
- Copilot inside apps provides contextual assistance
- Licensing determines the depth of functionality
- Understanding Work vs Web grounding is critical for AB-730
